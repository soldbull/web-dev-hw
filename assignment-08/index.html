How to re-braid hair Indigenous Style


Materials Needed:

  • Spray bottle with water
  • hairspray
  • comb

Directions:

  1. Start with the top braid if there is a top braid.
  2. Unravel the pre-existing braid all the way.
  3. Spray the hair with water before combing.
  4. Once the fly-aways are secured back into the hair, rebraid the hair.
  5. Repeat this process for the remaining braids.
  6. When all of the braids are re-braided, use hairspray to keep fly-aways down.
